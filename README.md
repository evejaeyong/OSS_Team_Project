# OSS_Team_Project 7조

## Title
Jaeyong's Love Finder

## Abstract
온라인 소개팅 사이트

자신의 정보를 등록하고 매칭이 되었을 때 상대방의 SNS 정보를 알려줌

로그인을 통해 사용자 정보를 저장하고 자신의 정보를 등록하거나 등록되어 있는 사람들의 정보를 볼 수 있다.
운명이라고 느껴진 분에게 매칭 신청을 보내면 상대방에게 이메일을 발송한다.
상대방이 수락했을 때 매칭이 되며, 상대방의 SNS 정보를 알 수 있다.
